Pengaturan Awal Admin Panel

Sebelum mulai, masukkan informasi GitHub kamu agar data tersimpan langsung ke repository GitHub — gratis dan permanen!

Langkah 1 — Buat Token GitHub
Buka github.com/settings/tokens/new
→ Beri nama token: SDMuh2Token
→ Centang repo (full control)
→ Klik Generate token → Copy token-nya
Langkah 2 — Isi Form di Bawah
Masukkan username GitHub, nama repository, dan token yang sudah dibuat.
Langkah 3 — Selesai!
Data akan tersimpan ke file data.json di repository GitHub kamu secara otomatis setiap kali kamu menyimpan konten.

⚙️ Konfigurasi GitHub

⚠️ Perhatian
Mengubah konfigurasi akan mempengaruhi semua koneksi ke GitHub. Pastikan data yang dimasukkan benar.
📋 Cara Buat Token Baru
  1. Buka github.com → Settings → Developer settings
  2. Klik Personal access tokens → Tokens (classic)
  3. Klik Generate new token (classic)
  4. Beri nama, centang repo, klik Generate
  5. Copy token (hanya tampil sekali!)
